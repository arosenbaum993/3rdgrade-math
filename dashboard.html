Connect to Google Sheet

You need two separate CSV URLs — one for each section tab. Follow these steps for each one.

  1. Open your Google Sheet with the simulation results
  2. Click File → Share → Publish to web
  3. Under the first dropdown, select "Gr3 Section 1" (not Entire Document)
  4. Under the second dropdown, select Comma-separated values (.csv)
  5. Click Publish and copy the URL — paste it below
  6. Repeat steps 3–5 for "Gr3 Section 2"
